# Lección 1: Introducción al Desarrollo Web
**Objetivo:** Comprender qué es un sitio web, cómo funciona y cuáles son sus componentes principales.

---

## ¿Qué es un sitio web?
Un **sitio web** es un conjunto de páginas accesibles desde Internet que pueden contener **texto, imágenes, videos, enlaces y otros recursos**.  
Se usan con fines educativos, informativos, comerciales o recreativos.

**Ejemplos de sitios web:**
- Wikipedia → Información enciclopédica  
- YouTube → Videos y contenido multimedia  
- Amazon → Compras en línea  
- Blogs personales → Publicación de experiencias o hobbies

> **Idea clave:** Todo sitio web combina **estructura (HTML)** y **estilo visual (CSS)**.

---

## Componentes principales de un sitio web

1. **HTML (HyperText Markup Language)**  
   - Lenguaje que define la **estructura y el contenido** de la página.

2. **CSS (Cascading Style Sheets)**  
   - Lenguaje que se utiliza para **dar estilo visual** al contenido de la página.  
   - Por ejemplo, permite cambiar **colores, fuentes, tamaños y alineación** de los elementos.  
   - En esta lección solo conoceremos el concepto; profundizaremos en CSS en la siguiente lección.

3. **Navegador web**  
   - Programa que interpreta HTML y CSS para mostrar la página visualmente.  

4. **Servidor y dominio (concepto básico)**  
   - El **servidor** almacena los archivos de la página web.  
   - El **dominio** es la dirección que escribes en el navegador (ej. `www.miweb.com`).
---

## Consejos

- HTML define la **estructura**, CSS define el **estilo**.  
- Guarda tus archivos con extensión `.html`.  
- Observa los cambios en tiempo real en el navegador.
